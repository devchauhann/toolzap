# toolzap
🚀 Excited to share my latest project: ToolZap - A Developer's Swiss Army Knife! 🛠️

ToolZap is a powerful, all-in-one developer toolbox that brings together essential utilities in one place. Built with pure HTML, CSS, and JavaScript (no frameworks!), it's lightweight and blazing fast.

🎯 Key Features:
• 🏞️ Image Tools - Resize, enhance, crop with custom filenames
• 📄 PDF Tools - Merge, split, convert formats
• 💻 Code Tools - Format, minify, beautify code
• 🔐 Security Tools - Hash generator, password generator, JWT decoder
• 📦 ZIP Tools - Compress, extract, encrypt files
• ⚡ And many more!

🔧 Tech Stack:
• Pure HTML5
• Modern CSS3
• Vanilla JavaScript
• Progressive Web App (PWA)
• Responsive Design
